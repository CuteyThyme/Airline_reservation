# Airline_reservation
基于C++ QT的航空订票系统   
可以实现航班的增删查改，航班查询、显示记录可以按照路程（时间）和费用进行关键字排序  
基本信息包括：航班号、航线、公司、额定票数、已订票人数、金额、用户等
